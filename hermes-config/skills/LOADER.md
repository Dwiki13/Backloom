# Hermes Skill Loader Rules

## RULE 1 — CORE ALWAYS AVAILABLE
CORE skills are always prioritized in every task.

## RULE 2 — TASK MATCHING
Select skills based on context:

- coding → software-development
- automation → devops
- trading/XAUUSD → trading-core
- writing/planning → productivity
- research → research

## RULE 3 — EDGE SKILLS
Only load EDGE skills if explicitly requested.

## RULE 4 — AVOID OVERLOAD
Never load more than 3 skill domains per task.

## RULE 5 — INDEX IS SOURCE OF TRUTH
Always refer to INDEX.md for classification.
