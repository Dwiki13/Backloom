# Prototype-First Outreach Approach

## Philosophy
Skip long WA intro templates. Directly send a prototype link + 1 sentence. People don't read long messages — they scan 2-3 lines. Proof (prototype) > 100 words of promises.

## Flow
1. **Scrape contacts** → 10-15 per day
2. **Send prototype link** → 1-2 minutes per contact
3. **If they respond** → Enter normal flow (ask needs, give price)
4. **If no response** → FU1 day 3, FU2 day 7
5. **Max 2 follow-ups** → Then archive

## Templates

### Intro (All Categories)
```
Halo, ini contoh website [kategori] yang pernah saya buat:
[LINK PROTOTYPE]

Kalau cocok, bisa saya buatkan yang serupa untuk [nama bisnis] 😊
```

### Follow Up 1 (Day 3)
```
Halo Kak, kemarin saya kirim contoh website untuk [nama bisnis]. Sempat dilihat?

Kalau berminat, bisa langsung saya bantu buatkan. Prosesnya cepat, 1-2 hari jadi 👍
```

### Follow Up 2 (Day 7, Final)
```
Halo Kak, ini follow up terakhir saya ya 😊

Kalau memang belum berminat, gapapa banget. Tapi kalau nanti butuh website, saya siap bantu.

Semoga [nama bisnis] makin sukses ya 🙏
```

### If They Ask Price
```
Harganya mulai Rp 1.500.000 (landing page 1 halaman, sudah termasuk desain responsive + tombol order via WA + domain .id 1 tahun + hosting 3 bulan).

Kalau butuh yang lebih lengkap (multi-halaman, galeri, SEO), ada paket Rp 2.500.000.

Kakak tertarik yang mana? 😊
```

### If They Say "Pikir-pikir"
```
Sip Kak, nggak perlu buru-buru. Kalau sudah siap, kabari saja ya 😊

Saya siap bantu kapanpun 👍
```

### If They Say "Mahal"
```
Kakak bisa mulai dari paket Starter dulu (Rp 1.500.000) — landing page 1 halaman, sudah termasuk desain responsive + tombol order via WA.

Kalau nanti butuh fitur lebih, bisa upgrade kapanpun 😊
```

## Comparison: Old vs New Approach
| Aspect | Old (Template) | New (Prototype) |
|--------|----------------|-----------------|
| Message length | 6-8 lines | 2-3 lines |
| Proof | None | Direct prototype link |
| CTA | "Gratis konsultasi" | "Kalau cocok, bisa saya buatkan" |
| Time per contact | 3-5 min | 1-2 min |
| Filter quality | Weak | Strong (responders = qualified) |
| Expected response | ~10% | 25-35% |

## Key Principles
1. **Personalize with business name** — "untuk [nama bisnis]" feels researched, not spam
2. **Don't explain features** — Let the prototype speak
3. **Don't offer discount** — Let them ask first
4. **Follow-up only with responders** — Don't waste time on non-responders
5. **Max 2 FU** — Then archive, don't be pushy
