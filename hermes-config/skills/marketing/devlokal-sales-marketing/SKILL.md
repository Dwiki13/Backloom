## Google Ads — Budget & Learning Phase

### ⚠️ Pause & Restart = Learning Phase Reset

Kalau campaign di-pause lama lalu di-restart, Google Ads akan masuk **learning phase** lagi (1-2 minggu). Dampak:
- CPC naik 10-30% selama learning
- Performance nggak stabil
- Budget terbuang untuk "belajar" ulang

**Rekomendasi:**
- Kalau mau stop, lebih baik **turunkan budget** (jangan pause total)
- Kalau terlanjur pause, kasih waktu 1-2 minggu untuk learning sebelum expect hasil
- **Jangan pause & restart berulang** — algoritma nggak pernah selesai belajar

### ⚠️ Budget Setting: Jangan Daily Budget = Target CPA

Kalau daily budget = target CPA, artinya max 1 conversion per hari. Google Ads butuh **minimal 5-10 conversions per minggu** untuk belajar dengan benar.

**Rekomendasi untuk budget terbatas:**
- Topup Rp 100k → daily budget Rp 25k (tahan 4 hari, bukan 2 hari)
- Target CPA tetap Rp 50k → max 2 conv/hari, total 4-8 conv dalam 4 hari

### ⚠️ Fokus Conversion Dulu, Conversion Value Nanti

Dengan budget terbatas (< Rp 500k):
- **Fokus ke conversion** (bukan conversion value)
- Set conversion value = harga paket terendah (Rp 1.500.000)
- Biarkan algoritma belajar dulu
- Evaluasi conversion value setelah dapat 10+ conversions

### 🔧 Conversion Tracking: Scroll-Depth Gating

**Masalah:** WA button click conversion tracking = inflated. Orang klik WA tapi nggak kirim pesan, atau kirim pesan tapi ghosting.

**Solusi:** Scroll-depth gating — conversion cuma fire kalau user udah scroll 40%+ (baca konten + harga) baru klik WA.

**Expected impact:**
- Conversion count turun (dari 36 → 15-20)
- TAPI conversion quality naik
- Cost per conversion lebih realistic

**Reference:** `references/scroll-depth-gating.md`

```python
briefing = """..."""  # your message text

payload = {
    "chat_id": -1003966561389,
    "message_thread_id": 1264,
    "text": briefing,
    "parse_mode": "Markdown"
}

payload_json = json.dumps(payload)
with open('/tmp/tg_payload.json', 'w') as f:
    f.write(payload_json)

result = subprocess.run(
    ['curl', '-s', '-X', 'POST',
     f'https://api.telegram.org/bot{token}/sendMessage',
     '-H', 'Content-Type: application/json',
     '-d', '@/tmp/tg_payload.json'],
    capture_output=True, text=True
)
print(result.stdout)
```
