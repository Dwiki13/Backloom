# Delivery Order Template Layout (XLSX)

Based on: Delivery Order HAS.xlsx (template from Drive)

## Header (rows 1-13, col F)
- Row 6: PT. HARMONI ARTHA SENTRA (bold)
- Row 9: Tagline
- Row 11-13: Address, city, phone

## Title (row 14)
- **DELIVERY ORDER** centered, bold

## Metadata (rows 16-19, col J/K)
- No DO, No PO, Tanggal PO, Tanggal Kirim

## Client Info (rows 21-23, col B)
- "Kepada Yth :" / Company name / "Up. [person]"

## Table (row 27+)
Headers: No (B) | Description (C) | Qty (M) | Uom (N)
Data starts row 28. No price columns.

## Sign-off
- "Pengirim" (left) | "Penerima" (right)

## Notes
- DO references a PO number
- No pricing — only quantities
- Format: DO/HAS/YYYY/MM/XXX
